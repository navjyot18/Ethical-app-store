# Data Sources

## 1. Exodus Privacy API
- **URL:** https://reports.exodus-privacy.eu.org/api
- **Cost:** Free
- **Rate Limit:** None documented
- **Data:** Trackers, Permissions
- **Reliability:** 95%

## 2. Google Play Store Scraper
- **Library:** google-play-scraper
- **Cost:** Free
- **Rate Limit:** ~100 requests/hour (unofficial)
- **Data:** App metadata, reviews, privacy policy URL
- **Reliability:** 90%
